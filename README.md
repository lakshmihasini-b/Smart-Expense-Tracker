# Smart-Expense-Tracker
💰 Smart Expense Tracker 

A modern and responsive Expense Tracker Dashboard built using HTML, CSS, and JavaScript. This application helps users manage their finances by tracking income and expenses, visualizing spending patterns, and storing data locally in the browser.

📌 Project Overview

The Expense Tracker Dashboard allows users to:

- Add and manage income and expense transactions.
- View total balance, total income, and total expenses.
- Categorize transactions for better organization.
- Analyze spending through interactive charts.
- Search and sort transaction history.
- Store data using Local Storage so information persists after refreshing the page.
  
✨ Features

💳 Transaction Management

- Add new income and expense transactions.
- Edit existing transactions.
- Delete individual transactions.
- Clear all transaction data with confirmation.
  
📊 Dashboard Analytics

- Total Balance calculation.
- Total Income summary.
- Total Expense summary.
- Real-time dashboard updates.
  
📈 Data Visualization

- Expense Category Pie Chart.
- Monthly Income vs Expense Bar Chart.
- Dynamic chart updates after every transaction.
  
🔍 Search & Filter

- Search transactions by title.
- Sort transactions by:
- Date (Newest First)
- Date (Oldest First)
- Amount (Highest First)
- Amount (Lowest First)
  
💾 Data Persistence

- Stores all transactions using Browser Local Storage.
- Automatically loads saved data on page refresh.
- No backend or database required.
  
🎨 User Interface

- Responsive design for desktop and mobile devices.
- Clean and modern dashboard layout.
- Smooth animations and transitions.
- Modal-based transaction form.
- Interactive cards and charts.

🛠 Technologies Used
- HTML5
- CSS3 
- JavaScript 
- Chart.js
- Browser Local Storage
